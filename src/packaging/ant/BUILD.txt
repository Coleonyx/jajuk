To compile Jajuk :

Requirements :
- JDK 1.7+
- Ant (download it on http://ant.apache.org/ )

-------
Build 
-------
$ ant

-> jajuk.jar is built in the 'bin' directory

------
Run
------
java -cp "lib/*:bin/jajuk.jar" org.jajuk.Main

------------
Junit tests
------------
$ unset DISPLAY  (to run tests headless)
$ ant tests
